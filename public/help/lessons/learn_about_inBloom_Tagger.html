learn about inBloom Tagger

The inBloom Tagger window is composed of a navigation bar at the top of the screen and three panes below the navigation bar. The user can use inBloom Tagger with limited functionality without signing in; however, for full functionality (including the ability to save the user’s work and publish the user’s learning resource metadata tags to the Learning Registry (LR)), the user must sign in with an inBloom username and password using the link at the top right of the screen.

Zoom

The following functions are available on the navigation bar (the Save and Publish buttons are available only to users who have signed in):

  • New. Create single new resources or multiple new resources.
  • Save. Save work in draft mode (signed-in users only).
  • Import. Import resources from a .csv file or .json file.
  • Export. Export resources in a .csv, .json, or .html file format.
  • Publish. Publish resource metadata to the Learning Registry (signed-in users only).
  • Help. Open the help system.
  • Report Issue. Open the reporting system.
  • CCSS Web. Opens a new tab to Corestandards.org, allowing users to search or browse the Common Core State Standards (including dot notations, descriptions, and URLs)

The Tabs pane determines the mode in which the user is working:

  • Click the Resources tab to select the resource(s) for which the user is entering information or saving. The user must select the check-box next to the relevant resources to designate the active resource for this tagging session.
  • Click the Output tab to confirm or review the user's work prior to saving the work out to an external file or publishing to the LR.
  • Click the History tab to view and/or modify the last 25 entries the user has submitted to the LR (signed-in users only).

The Detail pane displays the fields in which the tag data will be entered. The fields are grouped into three tabs, the General tab, Education tab, and the Alignment tab, on this pane.

The Resource pane displays the web page for the educational resource as specified by the URL entered on the General tab on the Detail pane.
