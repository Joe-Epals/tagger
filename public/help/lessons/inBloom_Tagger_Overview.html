inBloom Tagger Overview

The inBloom Tagger window is composed of a navigation bar at the top of the screen and three panes below the navigation bar. You can use inBloom Tagger with limited functionality without signing in; however, for full functionality (including the ability to save your work and publish your resources to the Learning Registry (LR)), you must sign in with an inBloom username and password using the link at the top right of the screen.

Zoom

The following functions are available on the navigation bar (the Save and Publish buttons are available only to users who have signed in):

  • New. Create single new resources or multiple new resources.
  • Save. Save your work in draft mode (signed-in users only).
  • Import. Import resources from a .csv file or .json file.
  • Export. Export resources in a .csv, .json, or .html file format.
  • Publish. Publish resources to the Learning Registry (signed-in users only).
  • Help. Open the help system.
  • Report Issue. Open the reporting system.
  • CSS Web. Opens a new tab to the Achievement Standards Network, allowing users to search or browse for the dot notation for a standard.

The Tabs pane determines the mode in which you are working:

  • Click the Resources tab to select the resource(s) for which you are entering information or which you are saving. You must select the check-box next to the relevant resources to designate the active resource for this tagging session.
  • Click the Output tab to confirm or review your work prior to saving your work out to an external file or publishing to the LR.
  • Click the History tab to view and/or modify the last 25 entries you submitted to the LR (signed-in users only).

The Detail pane displays the fields in which the tag data will be entered. The fields are grouped into three tabs on this pane, the General tab, Education tab, and the Alignment tab.

The Resource pane displays the web page for the educational resource as specified by the URL entered on the General tab on the Detail pane.
