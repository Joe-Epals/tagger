Import Multiple Resources from a CSV or JSON File

You can easily upload multiple resources from a .csv or .json file. Create the import file using the instructions found at: the following URL:

http://ec2-107-22-167-0.compute-1.amazonaws.com/LRMI%20Tagger%
20CSV%20Upload%20Template%20User%20Instructions.docx

Please note, you must save the completed upload file in either the .csv or .json format.

Open the Import window

Click Import.

Select the file to upload

Click Choose Files. The Browse dialog displays. Navigate to the file to be uploaded. Please note that the file must be saved in either a .csv or .json format.

Load the data

Click Load. The Resources are listed in the Resources pane. Select each resource (one at a time) and review the fields on the General, Education, and Alignment tabs to ensure that the upload was successful. Make any necessary changes. Changes are saved as you move your cursor from a field.
