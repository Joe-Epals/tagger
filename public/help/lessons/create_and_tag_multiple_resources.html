create and tag multiple resources

Many resources have multiple attributes that are common. By selecting multiple resources when you are creating the tags, all information that the user enters will be applied to all selected resources. The following instructions provide the steps for one method of tagging multiple resources.

1. Click New

Under the Resources tab, a new item displays.

2. Select the new item

3. Enter the Title and URL

On the General tab, enter the title and URL for the educational resource.

Title: Enter the title of the resource or component to be tagged (such as website name, book title, chapter title). Note: As the user clicks the cursor out of this field, the title of the resource updates in the Tag pane.

URL: Enter the web address of the page to be tagged. This page could be part of the resource itself or it could be a page describing the resource (such as a product page).

Description: Enter a description of the educational resource.

(Optional: Click Add Image to add an image of the educational resource. A thumbnail of the image will be created. In the inBloom Search application, the thumbnail displays in the search results when users search for the educational resource.

Note: Images should be in a square aspect ratio. Tagger will resize the image to the following resolutions:

  • 128 pixels x 128 pixels for inBloom Search
  • 64 pixels x 64 pixels for LRI Browser

If the user uploads an image that is different than a square aspect ratio, Tagger will crop it to a square aspect ratio.
If no image is provided, Tagger will attempt to display a screenshot of the webpage associated with the resource’s URL. Because this is an automated process, there is no guarantee what the software will capture. Therefore, uploading thumbnails is highly recommended. )

4. Add an additional resource

Click the New button.

5. Select the new resource

On the Resources tab, clear the checkmark from the first resource, and select the new resource. When you select the new resource and clear the checkmark from first resource, the fields in the Detail pane are cleared.

Note: The resource selected in the Resources tab controls the information that displays in the fields in the Details pane. Conversely, any information entered (or modified) in the Details pane will be saved for the item selected in the Resources tab. For example, if the Shapes resource is selected in the Resources tab, any information already entered for this resource will display in the fields in the Details pane, and any additional information entered or any modifications to the fields will be saved and associated with this resource. The user can view how this content is added or changed in the Output tab.

6. Enter the Title and URL for the new resource

On the General Tab, complete the form fields to provide general information about the second educational resource.

Title: Enter the title of the resource or component to be tagged (such as website name, book title, chapter title). Note: As the user clicks the cursor out of this field, the title of the resource updates in the Tag pane.

URL: Enter the web address of the page to be tagged. This page could be part of the resource itself or it could be a page describing the resource (such as a product page).

Description: Enter a description of the educational resource.

(Optional: Click Add Image to add an image of the educational resource. A thumbnail of the image will be created. In the inBloom Search application, the thumbnail displays in the search results when users search for the educational resource. Please see previous note regarding images for more information.)

Repeat steps 4-6 for each new resource to be added.

7. Enter the data common to all resources

On the Resources tab, select the resources for which to enter the common data.

Note that when multiple resources are selected, any fields that contain different values for different resources display blank. For example, the Title and URL fields should always display blank when multiple resources are selected.

Enter the information common to all resources in the Details pane on the General, Education, and Alignment tabs. For more information on entering information for a resource, please see the topic Tag a New Single Resource.

8. Enter the data specific to one resource or a subset of resources

On the Resources tab, select the resource(s) for which to enter the data.

Enter the information that is specific to the selected resource(s) on the General, Education, and Alignment tabs.

Repeat for each resource until all necessary information has been entered.
