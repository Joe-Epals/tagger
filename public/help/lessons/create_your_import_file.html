create your import file

You can bulk-upload educational resources into inBloom Tagger to create metadata tags that comply with the LRMI Specification.

Overview of the Import Process

1. Download the CSV Import Template to begin.

2. Use the following Data Structure Rules as you populate the template to ensure that your import file will load into inBloom Tagger without error. Please enter one resource per line.

Note: inBloom Tagger will only accept well-formed data; it is very important that your data follows these rules, or it will not import correctly.

3. Save the completed upload file in either the .csv or .json format.

4. Import the file.

Data Structure Rules

General Guidelines

  • All values can be any combination of uppercase or lowercase and still attempt to match the suggested values.
  • Any correctly formatted CSV should work.
  • All fields should allow for any UTF-8 characters.
