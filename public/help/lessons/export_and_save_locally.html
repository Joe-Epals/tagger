export and save locally

You may save output locally from an inBloom Tagger session in three formats

  • .csv file
  • .json file
  • .html file

Select the resources

On the Resources tab, select the resouces for which to save output.

Click Export.

Click Export. The Export sub-menu displays.

Select the appropropriate save option

From the Export sub-menu, select the appropriate option. Browse and specify the name and location to save the newly created output file.
