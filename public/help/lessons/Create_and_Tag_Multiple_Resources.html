Create and Tag Multiple Resources

Many resources have multiple attributes that are common. By selecting mutliple resources when you are creating the tags, all information that you enter will be applied to all selected resources. The following instructions provide the steps for one method of tagging multiple resources.

1. Click New

Under the Tag tab, a new item displays.

2. Select the new item

3. Enter the Title and URL

On the General tab, enter the title and URL for the educational resource.

Title: Enter the title of the resource or component to be tagged (such as website name, book title, chapter title). Note: As you move your cursor from this field, the title of the resource updates in the Tag pane.

URL: Enter the web address of the page to be tagged. This page could be part of the resource itself or it could be a page describing the resource (such as a product page).

4. Add an additional resource

Click the New button.

5. Select the new resource

Clear the checkmark from the first resource, and select the new resource. When you select the new resource and clear the checkmark from first resource, the fields in the Detail pane are cleared.

Note: The resource selected in the Tags tab controls the information that displays in the fields in the Details pane. Conversely, any information entered (or modified) in the Details pane will be saved for the item selected in the Tags tab. For example, if the Shapes resource is selected in the Tags tab, any infomation already entered for this resource will display in the fields in the Details pane, and any additional information entered, or any modifications to the fields will be saved and associated with this resource. You can view this how this content is added or changed while you are working in the Output tab.

6. Enter the Title and URL for the new resource

On the General Tab, complete the form fields to provide general information about the second educational resource.

Title: Enter the title of the resource or component to be tagged (such as website name, book title, chapter title). Note: As you move your cursor from this field, the title of the resource updates in the Tag pane.

URL: Enter the web address of the page to be tagged. This page could be part of the resource itself or it could be a page describing the resource (such as a product page).

Repeat steps 4-6 for each new resource to be added.

7. Enter the data common to all resources

Select the resources for which to enter the common data.

Note that when multiple resources are selected, any fields that contain different values for different resources display blank. For example, the Title and URL fields should always display blank when multiple resources are selected.

Enter the information common to all resources in the Details pane on the General, Education, and Alignment tabs. For more information on entering information for a resource, please see the topic Tag a New Single Resource.

8. Enter the data specific to one resource or a subset of resources

Select the resource(s) for which to enter the data.

Enter the information that is specific to the selected resource on the General, Education, and Alignment tabs.

Repeat for each resource until all necessary information has been entered.
