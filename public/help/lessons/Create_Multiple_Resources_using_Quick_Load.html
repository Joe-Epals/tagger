Create Multiple Resources using Quick Load

You can efficiently create multiple resources in a single action using the Quick Load feature. Once the resources are created, you may edit them to add additional information.

Open the Quick Load dialog

Click Import. The Quick Load dialog displays.

Enter the Title and URL for each of the resources to be created

Enter the Title and URL of the resource to be tagged in the following format:

Title,URL

A proper entry for multiple items would look like the following:

Sample 1 title,http://www.educationwebsite1.com
Sample 2 title,http://www.educationwebsite2.com
Sample 3 title,http://www.educationwebsite3.com
Sample 4 title,http://www.educationwebsite4.com

The Title is the name of the resource or component to be tagged, such as a website name, book title, or chapter title.
The URL is the web address of the page to be tagged (such as http://www.educationwebsite.com). This page could be part of the resource itself, or it could be a page describing the resource (for example, a product page).

If you have a list of multiple resources to tag, enter each resource on a new line.

Load the data

Click Load. The newly created resources are listed in the Tag pane.

Enter the data common to all resources

Select the resources for which to enter the common data.

Note that when multiple resources are selected, any fields that contain different values for different resources display blank. For example, the Title and URL fields should always display blank when multiple resources are selected.

Enter the information common to all resources in the Details pane on the General, Education, and Alignment tabs. For more information on entering information for a resource, please see the topic Tag a New Single Resource.

Enter the data specific to one resource or a subset of resources

Select the resource(s) for which to enter the data.

Enter the information that is specific to the selected resource(s) on the General, Education, and Alignment tabs.

Repeat for each resource until all necessary information has been entered.
