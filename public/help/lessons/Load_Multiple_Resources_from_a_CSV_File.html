Load Multiple Resources from a CSV File

You can easily upload multiple resources from a .csv file. Download the .csv template from http://23.23.213.206/LRMI%20template.csv

Please note, you must save the completed upload file in the .csv format.

Load the file

Click Load. The Load dialog displays.

Select the file to upload

Click Choose Files. The Browse dialog displays. Navigate to the file to be uploaded. Please note that the file must be saved in a .csv format.

Load the data

Click Load. The Resources are listed in the Tag pane. Select each resource (one at a time) and review the fields on the General, Education, and Alignment tabs to ensure that the upload was successful. Make any necessary changes. Changes are saved as you move your cursor from a field.
