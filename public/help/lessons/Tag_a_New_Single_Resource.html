Tag a New Single Resource

Follow the instructions below to create a tag for your educational resource. The more detail that you provide about your resource, the more visible it will be to users searching for resources.

1. Click New

Under the Resources tab, a new item displays.

2. Ensure that the item is selected.

3. Enter the General Information

On the General Tab, complete the form fields to provide general information about the educational resource.

Title: Enter the title of the resource or component to be tagged (such as website name, book title, or chapter title).

URL: Enter the web address of the page to be tagged. This page could be part of the resource itself or it could be a page describing the resource (such as a product page).

Language: Select the primary language in which the resource or component is published.

Created On: Click the field and use the calendar to select the date on which the resource or component was created, or type the date in the YYYY-MM-DD format.

Topic/Subject: Enter the topic or subject of the resource or component to be tagged (such as Geography, Measurement, or Geometry).

Created By: Enter the individual credited with creating the resource or component (such as the name of the author, programmer, director, or professor).

Usage Rights URL: Enter the URL where the owner specifies permissions for using the resource (such as http://creativecommons.org/licenses/by/3.0).

Publisher: Enter the organization credited with publishing the resource or component (such as the name of the publishing house, company, or organization).

Is Based on URL: This is required if the resource is based on another resource. Enter the resource used in the creation of this resource.

4. Click the Education tab

Click the Education tab. Note the sub-headings beneath the Education tab: click the subheadings to reveal and hide the fields related to the subheadings.

5. Specify the intended end user role(s)

Click End User to reveal the possible options. Select the individual or group for which the resource was produced (such as student, teacher, or administrator). Multiple roles can be selected: hold the Shift key down while selecting multiple adjacent roles or hold the CTRL key down to individually select mutliple roles. Use the scroll bar to view additional roles.

6. Select the typical age range(s)

Click Typical Age Range to reveal the possible options. Select the age range of the resource's intended end user. Multiple roles can be selected: hold the Shift key down while selecting multiple adjacent roles or hold the CTRL key down to individually select mutliple roles. Use the scroll bar to view additional age ranges.

Please note that as LRMI evolves, this is an area where we may see changes in how the schema is structured and consequently in how typical age range is input.

7. Specify the educational use(s)

Click Educational Use to reveal the possible options. Select the option that best describes how the resource will be used in an educational setting. Multiple uses can be selected: hold the Shift key down while selecting multiple adjacent uses or hold the CTRL key down to individually select mutliple uses. Use the scroll bar to view additional educational uses.

8. Specify the interactivity type

Select the predominant mode of learning supported by the resource. Multiple types can be selected: hold the Shift key down while selecting multiple adjacent types or hold the CTRL key down to individually select mutliple types. Use the scroll bar to view additional interactivity types.

9. Specify the instructional format of the learning resource

Select the instructional format(s) in which the resource is made available to the intended end user. Multiple types can be selected: hold the Shift key down while selecting multiple adjacent types or hold the CTRL key down to individually select mutliple types. Use the scroll bar to view additional learning resource types.

10. Specify the media format

Select the media format(s) in which the resource or component is published or produced. Multiple types can be selected: hold the Shift key down while selecting multiple adjacent types or hold the CTRL key down to individually select mutliple types. Use the scroll bar to view additional media types.

11. Enter the time required (optional)

Enter the estimated or typical amount of time it takes to work through or complete the objectives in the resource by clicking and dragging the slider bar to the desired amount of time. Please note that we use this specific control to ensure that Tagger input conforms to the ISO 8601 data elements and interchange formats to define the duration as specified in the LRMI Specification.

12. Enter the learning standard, competency, or learning outcome to which this resource is aligned

Click the Alignment tab.

13. Click Add Alignment

14. Specify the set of standards to which the resource is aligned

Select the set of standards to which the resource is aligned. The 1.1.0 release currently only supports the Common Core State Standard educational framework. Tagger is designed to be extensible to support other educational frameworks.

15. Select the Alignment Type

From the Alignment Type list, select the way in which the resource is aligned with the standard.

16. Enter the standard to which this resource is aligned

In the Dot Notation field, enter the standard in dot notation format. If you selected Common Core Standards, as you begin typing, possible matches will display in the drop-down box beneath the field. Select the correct standard.

The entry in this field will be reflected in the Name property of the Alignment Object from the LRMI specification 1.0. If the Educational Alignment type selected is Common Core State Standards, the Item URL field will populate from Corestandards.org, and the Description field will populate automatically with information from the Achievement Standards Network.

Zoom

Note: If you do not know the dot notation for the standard, you can use the CCSS Web button in the top right of the main window to open a new tab to the Achievement Standards Network and search or browse to the appropriate standards.

17. Click Add

Click Add Alignment to Session to create the alignment. The new alignment displays in the list in the Alignment pane.

18. (optional) Create additional alignments

To create additional alignments, repeat steps 13-17.
