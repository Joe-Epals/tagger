Tag a New Single Resource

Follow the instructions below to create a tag for your educational resource. The more detail that you provide about your resource, the more visible it will be to users searching for resources.

1. Click New

Under the Tag tab, a new item displays.

2. Select the new item

3. Enter the General Information

On the General Tab, complete the form fields to provide general information about the educational resource.

Title: Enter the title of the resource or component to be tagged (such as website name, book title, chapter title).

URL: Enter the web address of the page to be tagged. This page could be part of the resource itself or it could be a page describing the resource (such as a product page).

Language: Select the primary language in which the resource or component is published.

Created On: Click the field and use the calendar to select the date on which the resource or component was created, or type the date in the YYYY-MM-DD format.

Topic/Subject: Enter the topic or subject of the resource or component to be tagged (such as Geography, Measurement, or Geometry).

Created By: Enter the individual credited with the creating the resource or component (such as the name of the author, programmer, director, or professor).

Usage Rights URL: Enter the URL where the owner specifies permissions for using the resource (such as http://creativecommons.org/licenses/by/3.0).

Publisher: Enter the organization credited with publishing the resource or component (such as the name of the publishing house, company, or organization).

Is Based on URL: This is required if the resource is based on another resource. Enter the resource used in the creation of this resource.

4. Click the Education Tab

Click the Education tab. Note the sub-headings beneath the Education tab: click the subheadings to reveal and hide the fields related to the subheadings.

5. Specify the intended end user role(s)

Click End User to reveal the possible options. Select the individual or group for which the resource was produced (such as student, teacher, or administrator). Multiple roles can be selected: hold the Shift key down while selecting multiple adjacent roles or hold the CTRL key down to individually select mutliple roles. Use the scroll bar to view additional options. You are not limited to the suggested vocabulary. If your intended end user role is not available, you can use the Other field to enter in whatever text you would like in this section.

6. Select the typical age range(s)

Click Typical Age Range to reveal the possible options. Select the age range of the resource's intended end user. Multiple roles can be selected: hold the Shift key down while selecting multiple adjacent roles or hold the CTRL key down to individually select mutliple roles. Use the scroll bar to view additional options. You are not limited to the suggested age ranges. If your intended age range is not available, you can use the Other field to enter the appropriate age range(s) in this section.

Please note that as LRMI evolves, this is an area where we may see changes in how the schema is structured and consequently in how typical age range is input.

7. Specify the educational use(s)

Click Educational Use to reveal the possible options. Select the option that best describes how the resource will be used in an educational setting. Multiple uses can be selected: hold the Shift key down while selecting multiple adjacent uses or hold the CTRL key down to individually select mutliple uses. Use the scroll bar to view additional options. If your educational use is not available, you can use the Other field to enter the appropriate educational use(s) in this section.

8. Specify the interactivity type

Select the predominant mode of learning supported by the resource. Multiple types can be selected: hold the Shift key down while selecting multiple adjacent types or hold the CTRL key down to individually select mutliple types. Use the scroll bar to view additional options.

9. Specify the instructional format of the learning resource

Select the instructional format(s) in which the resource is made available to the intended end user. Multiple types can be selected: hold the Shift key down while selecting multiple adjacent types or hold the CTRL key down to individually select mutliple types. Use the scroll bar to view additional options. If your learning resource type is not available, you can use the Other field to enter the appropriate learning resource(s) in this section.

10. Specify the media format

Select the media format(s) in which the resource or component is published or produced. Multiple types can be selected: hold the Shift key down while selecting multiple adjacent types or hold the CTRL key down to individually select mutliple types. Use the scroll bar to view additional options. If the correct media format is not available, you can use the Other field to enter the appropriate media format(s) in this section.

11. Enter the time required (optional)

Enter the estimated or typical amount of time it takes to work through or complete the objectives in the resource by clicking and dragging the slider bar to the desired amount of time. Please note that we use this specific control to ensure that Tagger input conforms to the ISO 8601 Data elements and interchange formats to define the duration as specified in the LRMI Specification.

12. Enter the learning standard, competency, or learning outcome to which this resource is aligned

Click the Alignment Tab.

13. Specify the set of standards to which the resource is aligned

Select the set of standards to which the resource is aligned. The 1.0 release currently only supports the Common Core State Standard educational framework. Tagger is designed to be extensible to support other educational frameworks.

14. Select the Alignment Type

From the Alignment Type list, select the way in which the resource is aligned with the standard.

15. Enter the standard to which this resource is aligned

In the Dot Notation field, enter the standard in dot notation format. If you selected Common Core Standards, as you begin typing, possible matches will display in the drop-down box beneath the field. Select the correct standard.

The entry in this field will be reflected in the Name property of the Alignment Object from the LRMI specification 1.0. If the Educational Alignment type selected is Common Core State Standards, the Item URL field will populate from Corestandards.org, and the Description field will populate automatically with information from the Achievement Standards Network.

Note: If you do not know the dot notation for the standard, you can use the CCSS Web button in the top left to open a new tab to the Achievement Standards Network and search or browse to the appropriate standards.

16. Click Add

Click Add to create the alignment. The Alignment displays in the list near the bottom of the pane.

17. (optional) Create additional alignments or delete alignments

To create additional alignments, repeat steps 13-16.

To delete an alignment, select the alignment from the Previous Session Alignments list and click Delete. This will remove that specific standard from the output tab and the output file, but leave that standard in the previous alignments list for use in the current tagging session.
